주요 프로젝트

Triplet

프로젝트 개요

여행 예산 계획과 여행 결제 내역을 카테고리 별로 분류해 예산 관리를 돕고, 다른 사람의 여행 예산 사용 내역을 참고할 수 있어 예산 계획에 도움을 주는 여행 예산 관리 애플리케이션 입니다.

Spring Boot와 React.js, PWA 를 활용하여 웹 애플리케이션을 개발했습니다.

역할 : 팀장, 인프라, 백엔드

주요 개발

  • EC2 Linux 환경에서 Jenkins, GitLab, Docker를 활용한 CI/CD 파이프라인 구축
  • 외부 API를 활용한 환율 조회 및 환전 기능 구현
  • Redis를 활용한 환율 데이터 캐싱 및 Spring 스케줄러로 자동 갱신
  • Lua Script로 환율 삭제 및 삽입을 트랜잭션으로 묶어 원자성 보장

🗺️ 여행 계좌 개설 및 관리

📱 QR 간편 결제 기능

💱 환전 및 외화 계좌 관리

👥 모임 통장 기능 제공

📊 카테고리별 예산 설정

💳 결제 시 카테고리 자동 분류 및 실시간 지출 내역 확인

📤 여행 예산 내역 공유 기능

깃허브 이동

SSAFIT MALL

프로젝트 개요

중고거래 사기 피해 예방을 위한 헬스 용품 및 운동 보조제 중고 거래 쇼핑몰 사이트입니다.

Spring Boot 를 사용해 구현한 REST API 와 Vue.js 로 구현한 클라이언트로 구성된 Single Page Application 입니다.

역할 : 팀장

주요 개발 : Full-Stack

  • DB 테이블 설계
  • 깃 프로젝트 관리
  • 아래 기능의 REST API 설계와 Axios 비동기통신 구현

      • 📝 판매자 상품 CRUD
