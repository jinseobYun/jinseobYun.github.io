SSAFIT MALL

프로젝트 개요

중고거래 사기 피해 예방을 위한 헬스 용품 및 운동 보조제 중고 거래 쇼핑몰 사이트입니다.

Spring Boot 를 사용해 구현한 REST API 와 Vue.js 로 구현한 클라이언트로 구성된 Single Page Application 입니다.

역할 : 팀장

주요 개발 : Full-Stack

  • DB 테이블 설계
  • 깃 프로젝트 관리
  • 아래 기능의 REST API 설계와 Axios 비동기통신 구현

      • 📝 판매자 상품 CRUD

      💳 구매자 상품 조회 및 주문, 카카오 페이 결제

      ❤️ 관심 상품 기능

      🛠️ 관리자 페이지 ( 유저, 상품 관리 기능 )

      🤖 챗 지피티 API 챗봇 기능
