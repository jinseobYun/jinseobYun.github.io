✔️ 프로젝트 개요

목표

여행 중 지출을 효율적으로 관리하고, 다른 여행자들의 데이터를 참고하여 예산 계획을 쉽게 세울 수 있도록 돕는 여행 예산 관리 애플리케이션입니다.

기획 배경

여행을 처음 준비하는 사람들이 예상 비용을 파악하는 데 어려움을 겪고, 여행 중 예산 초과나 지출 내역을 기억하지 못하는 문제가 빈번하게 발생합니다. 최신 정보의 부족과 관리의 어려움으로 인해 발생하는 불편을 해결하기 위해 기획되었습니다.

주요 기능

다중 통화 계좌 관리, 여행 계좌 개설, QR 간편 결제, 원화 송금 및 환전, 모임 통장, 카테고리별 예산 설정, 실시간 예산 사용 내역 저장, 여행지 및 예산 추천, 자동 분류 및 실시간 지출 확인, 여행 예산 내역 공유

✔️ 담당 역할 및 기여

역할

팀장, 인프라 및 백엔드 개발 담당

프로젝트 기여

CI/CD 파이프라인 구축, Docker 및 클라우드 환경 배포, 외부 API를 활용한 환율 정보 조회 및 환전 API 구현, Redis를 활용한 환율 데이터 캐싱 및 Lua Script를 통한 원자성 보장

✔️ 사용 기술/이유

사용 기술

Spring Boot, Docker, Redis, Jenkins, GitLab, AWS EC2, Ubuntu, Spring Data JPA, MySQL

사용 이유

Docker: 일관된 환경에서 애플리케이션을 배포하기 위해
Redis: 빠른 데이터 조회
Lua Script: 원자성 보장

✔️ 구현 사항

핵심 기능

사용자의 나이, 성별, 여행 국가 등을 기반으로 다른 사용자가 공유한 예산 데이터를 제공하고, 자동으로 결제 내역을 카테고리별로 분류하여 관리. 여행 중 발생하는 지출 내역을 실시간으로 저장하고, 환율 변동에 대응해 자동으로 갱신.

Lua Script를 활용한 Redis 데이터의 원자성 보장
Jenkins와 GitLab을 통한 자동화된 CI/CD 파이프라인 구축

✔️ 문제 해결 사례 선택사항

문제

외부 API를 통해 환율 데이터를 받아와 Redis에 저장하는 과정에서, Redis가 단일 스레드 기반으로 작동하여 개별 명령어의 원자성은 보장되지만, 복수의 명령어를 순차적으로 실행할 때 완전한 원자성이 보장되지 않을 수 있다는 점을 인식했습니다. 우리가 만드는 프로젝트가 핀테크 서비스를 다루고 있어 결제, 송금, 충전 등 실제 돈이 오가는 시스템이기 때문에, 정확한 환율 데이터를 제공하는 것이 매우 중요하다고 생각했습니다. 이런 특성상, 조금의 오류 가능성도 배제할 수 없다고 생각했습니다.

시도한 방법

이를 해결하기 위해 Lua Script를 사용해 환율 삭제와 삽입 명령어를 하나의 트랜잭션으로 묶어 데이터 일관성과 원자성을 확보했습니다. 이 과정에서 데이터 무결성을 지키는 방법을 깊이 고민하며 구현할 수 있었습니다.

✔️ 결과 및 성과

성과

EC2 Ubuntu 환경에 서비스를 배포하며 Linux의 기본적인 명령어에 익숙해졌고, Docker를 통해 컨테이너들을 관리하며 Docker를 이해할 수 있었습니다.

또한 Jenkins를 활용해 CI/CD 파이프라인을 구축함으로써 배포 자동화에 대한 개념을 익히고 이를 직접 구현할 수 있는 역량을 갖추게 되었습니다.

✔️ 회고

이번 프로젝트를 통해 팀장으로서뿐만 아니라 인프라 구축을 담당한 개발자로서, 내가 맡은 역할을 책임지고 완수하는 데 큰 의미를 느꼈고, 팀이 작성한 코드를 실제 서비스할 수 있는 환경으로 만들어낸 경험은 뿌듯함과 동시에 큰 책임감을 다시 한 번 길러주었습니다. 어려움 속에서도 포기하지 않고 해결해낸 과정이 제게 많은 성장을 가져다주었습니다.

Spring Boot와 React.js, PWA를 활용하여 웹 애플리케이션을 개발했습니다.

역할 : 팀장, 인프라, 백엔드

주요 개발

  • EC2 Linux 환경에서 Jenkins, GitLab, Docker를 활용한 CI/CD 파이프라인 구축
  • 외부 API를 활용한 환율 조회 및 환전 기능 구현
  • Redis를 활용한 환율 데이터 캐싱 및 Spring 스케줄러로 자동 갱신
  • Lua Script로 환율 삭제 및 삽입을 트랜잭션으로 묶어 원자성 보장

SSAFIT MALL

프로젝트 개요

중고거래 사기 피해 예방을 위한 헬스 용품 및 운동 보조제 중고 거래 쇼핑몰 사이트입니다.

Spring Boot를 사용해 구현한 REST API와 Vue.js로 구현한 클라이언트로 구성된 Single Page Application입니다.

역할 : 팀장, Full-Stack

주요 개발

  • DB 테이블 설계
  • 깃 프로젝트 관리
  • 아래 기능의 REST API 설계와 Axios 비동기 통신 구현

📝 판매자 상품 CRUD

💳 구매자 상품 조회 및 주문, 카카오 페이 결제

❤️ 관심 상품 기능
